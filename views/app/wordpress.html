WORDPRESS:

Wat is WordPress precies? WordPress is het populairste content management systeem (CMS) ter wereld.Meer dan 30% van alle websites op aarde gebruikt WordPress. En meer dan 60% van alle websites die een CMS gebruiken, gebruikt WordPress. De nummer twee, Joomla, heeft een marktaandeel van ongeveer 5%. Dit betekent dat WordPress twaalf keer zo populair is dan de nummer twee.Het is vrij eenvoudig om een eigen WordPress website te maken. Ook voor totale beginners. Dit komt omdat WordPress werkt templates (thema’s) en plugins. Je bouwt dus niet een website vanuit het niets op. Je kiest een template die bij jou past en die pas je naar wens aan.Om WordPress echt te begrijpen, moeten we de drie bouwstenen van WordPress bespreken. Plugins zijn ‘kleine’ blokjes functionaliteit. Het zijn extra opties die je aan jouw site kan toevoegen. Wil je een contactformulier? Dan kan je daar een plugin voor installeren. Wil je deelknoppen? Installeer een plugin? Hulp bij SEO? Een plugin. Iets dat spam tegenhoudt? Een plugin. Betalingen via iDeal? Daar is een plugin voor.Voor alles wat je nodig hebt is er wel een plugin te vinden.Zo kan je met de plugin WooCommerce een hele webshop toevoegen aan jouw website. In combinatie met een plugin voor een leermodule kan je een cursussen aanbieden. Om WordPress te gebruiken moet je een hostingpakket en een domeinnaam hebben.Hosting is het huren van een stukje schijfruimte op een server die 24/7 verbonden staat met het internet. Om WordPress te gebruiken, moet je de WordPress bestanden op dat stukje schijfruimte zetten. Dit klinkt misschien ingewikkeld, maar dat valt reuze mee. De meeste webhosts (bedrijven die hosting verkopen) bieden een one-minute WordPress installatie aan. Wij (Matrix) hebben zelf Wordpress hosting waarbij WordPress automatisch geïnstalleerd wordt en waar je ondersteuning krijgt bij WordPress gerelateerde vragen).Na de (automatische) installatie kan je direct beginnen. Met WordPress kan je overal waar je internet hebt, jouw website, blog of webshop bewerken. Na het inloggen beland je op het admin dashboard van jouw WordPress website. Via het dashboard heb je de controle om jouw hele website aan te passen.
